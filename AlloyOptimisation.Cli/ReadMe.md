# Alloy optimisation CLI

This project implements a simple CLI tool to use the Alloy optimiser.

## Running the tool

Navigate to the AlloyOptimisation.Cli folder and run:
```
dotnet run <your_cost_constraint>
```
e.g. `dotnet run 18`

Example output:
```
Optimal alloy composition:
  Cr: 22.00
  Co: 10.00
  Nb: 1.10
  Mo: 6.00
  Ni: 60.90

Creep Resistance: 1.729988E+018
Total Cost:       17.98
```

Running without any parameter will print tool usage.

## Making an executable

An executable can be generated by running:
```
dotnet publish -c Release -r win-x64 --self-contained true /p:PublishSingleFile=true
```

This should generate a single executable file, at location `AlloyOptimisation.Cli\bin\Release\net10.0\win-x64\publish`.
Copy files `AlloyOptimisation.Cli.exe` and `NickelAlloyComponentConfig.json` to a desired location and run the application from there with:
```
.\AlloyOptimisation.Cli.exe 18
```
**Please note**: this method is for Windows only.
